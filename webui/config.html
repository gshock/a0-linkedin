The plugin stores separate personal and organization LinkedIn profiles. In chat, the agent should use the correct one based on your prompt. If you do not specify which mode to use, the agent should ask.

Use this profile for posting under your own LinkedIn identity.

Typical scopes by mode:
  • w_member_social for personal posting
  • r_member_social for personal post reading if supported
  • w_organization_social for organization posting after approval
  • r_organization_social for organization post retrieval after approval
